# Fylo Data Storage Component

This project showcases a data storage component with a two-column layout, features section, and footer. It's built using HTML and CSS to create an attractive and functional web page.

## Preview

![Preview](./Preview.png)

You can see the live preview of this project [here](https://ahmedhanye.github.io/Data-Storage-Component/).

## Project Structure

- `index.html`: Main HTML file defining the page structure.
- `CSS/normalize.css`: CSS file for normalizing styles across browsers.
- `CSS/main.css`: Main CSS file styling the web page.
- `images/`: Directory containing project images.

## How to Run Locally

To run the project on your local machine:

1. Clone this repository.
2. Open the `index.html` file in your web browser.

Enjoy exploring the Fylo Data Storage Component!
